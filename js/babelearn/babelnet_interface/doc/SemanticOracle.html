Class: SemanticOracle

SemanticOracle()

Implements a facade towards the BabelProxy class, in order to decouple the business logic of the game from the details of the BabelNet/Babelfy API (e.g. existance of synsets, synset IDs and so on).

getImages(word, language, meaningPosition) → {Array.<string>}

Retrieves a list of image URLs associated to the i-th meaning of a given word in a given language.
Parameters:
Name Type Description
word string The word whose associated image URLs will be searched.
language word The language the given word belongs to. Use two letters abbreviations, e.g. 'EN' for English.
meaningPosition number The position of the meaning in the list of meanings of the given word, e.g. settings this to 1 allows to retrieve the images associated to the first meaning of the word. The order of the meanings, and so which one is the i-th meaning of a word, depends on the underlying BabelNet API, so the position is only intended to be used to retrieve different meanings of the same word, by calling this method with the same word and language but changing the position. This number must be a non-negative integer (positions are zero-based).
